Detailed Description

a data structure to describe an object to work on.
the object can be one or more:

  • atom(s)
  • chemical species
  • coordination sphere(s) (total or partial)
  • fragment(s)
  • molecule(s)
  • selection(s) of atom(s) from any project in the workspace
